# Dashboard Executivo de Portfólio de Projetos

## Objetivo
Analisar o portfólio de projetos da empresa sob a ótica financeira, execução e cronograma, apoiando a tomada de decisão.

## Principais Análises
- Visão executiva com indicadores estratégicos
- Gestão financeira dos projetos
- Execução e capacidade operacional
- Cronograma e acompanhamento de prazos

## Ferramentas
- Power BI
